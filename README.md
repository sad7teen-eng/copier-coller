# Copie Rapide

Application web pour sauvegarder et copier des textes en un clic.  
Fonctionne sur **mobile** et **ordinateur**, sans installation.

## Comment l'utiliser

1. Appuie sur **Nouveau**
2. Colle ton texte (et un titre optionnel)
3. Appuie sur **OK**
4. Appuie sur une carte pour **copier le texte** automatiquement

Les textes sont sauvegardés localement dans le navigateur (localStorage).

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Entrée` | Valider (OK) |
| `Échap` | Annuler |

## Héberger sur GitHub Pages

1. Crée un repo GitHub (ex: `copie-rapide`)
2. Push ce dossier
3. Va dans **Settings → Pages → Branch: main → Save**
4. Ton app sera dispo sur `https://ton-pseudo.github.io/copie-rapide`

## Technologie

HTML + CSS + JavaScript vanilla — aucune dépendance, un seul fichier.
